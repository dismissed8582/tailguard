# Archived result images

The PNG files in this directory were inherited with the repository. They are
not tied to a tracked input snapshot, complete run configuration, dependency
and solver versions, or an automated command that regenerates these exact
files, and they predate the validated implementation in `tailguard/`.

They are retained only as historical artifacts. They must not be used to
support numerical, performance, tail-dependence, validation, or sourcing
claims. New results should be generated from a documented input data source and
configuration, with a reproducible script and an independent review.

See the [legacy-artifact audit](../LEGACY_ARTIFACT_AUDIT.md) for the source and
mathematical-integrity review of the inherited PDF and these figures.
